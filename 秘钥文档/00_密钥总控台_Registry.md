# 🔐 密钥总控台 (Key Registry)

> ⚠️ **安全提醒**: 此文件仅供 AI 助手内部查询，绝不应提交到公共 Git 仓库。建议将本目录加入 `.gitignore`。

## 📌 使用法则

AI 助手在编码时需要任何密钥/Token/Key，**必须先查本文件**，再用文件读取工具获取对应密钥文件中的实际值。**绝不向用户索要已存在于此的密钥。**

## 📂 密钥文件模板

每个密钥文件应包含：
```markdown
# [服务名] 密钥

**用途**: [这个密钥用于什么]
**创建时间**: YYYY-MM-DD
**过期时间**: [如适用]

## 密钥值
- **API Key**: `your-api-key-here`
- **Secret**: `your-secret-here`

## 使用说明
[如何在代码中使用此密钥]
```
